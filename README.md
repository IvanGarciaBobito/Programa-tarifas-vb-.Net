# Programa tarifas vb .Net

Programa creado hace unos años para poder cobrar los servicios manualmente a los clientes cuando el sistema principal no estaba disponible, este programa
lo desarrolle por voluntad propia para hacernos el trabajo mas facil y agilizar los servicios frente a imprevistos.

El programa muestra una lista con el precio por minuto hasta el maximo preestablecido por el negocio.

El programa lleva la cantidad por tramo y el tramo de tiempo predefinidos por el negocio.

Se incorpora un metodo de busqueda por hora y minutos para agilizar la consulta
